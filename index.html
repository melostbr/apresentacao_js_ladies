Tipos de Dados

  • number

    Pra essa linguagem, número é número, seja ele um número inteiro ou decimal.

  • string

    Qualquer texto (ou aquilo que você escrever entre aspas) é definido como string.
    "Eu ♥ Javascript"

  • boolean

    Famosa "lógica booleana": true (verdadeiro) ou false (falso)

  • null

    Quando queremos definir um valor "nulo" ou vazio

  • undefined

    Quando não há um tipo definido. Na maioria das vezes, não difere muito do null.

Operadores Lógicos

As vezes, precisamos fazer algumas validações lógicas pra realizarmos decisões no nosso código

  • Comparações 'simples"

    console.log(3 > 2) //True
    console.log(3 < 2) //False

    É possível comparar duas strings, mas lembre-se que o javascript analisa cada caractere, da esquerda pra direita.

    console.log("a" > "z") //False
    console.log("pixar" > "disney") //True

  • Operador ternário

    Ele analisa o primeiro parâmetro e "age" conforme o resultado for true (verdadeiro) ou false (falso)

    console.log(true ? 1 : 2); // → 1
    console.log(false ? 1 : 2); // → 2

Estruturas de Controle

Em alguns momentos precisaremos permitir que o nosso programa decida qual pra qual "caminho" ele deverá ir.

Se a nossa regra de negócio exige que, dependendo de determinado resultado, o nosso programa execute uma tarefa diferente da esperada, estamos realizando um controle. As principais formas de realizar esse feito é atraves do if/else,while, for ou até mesmo o "operador ternário" que vimos anteriormente

var nome = 'gatinhos';
if (nome == 'cachorrinhos') {
  nome += ' fofinhos';
} else if (nome == 'gatinhos') {
  nome += ' dominam os humanos';
} else {
  nome += '!';
}
nome == 'gatinhos dominam os humanos'; //true

Operações Matemáticas

Podemos realizar operações matemáticas simples com os seguintes operadores:

  • soma: 2 + 2 = 4
  • subtração: 2 - 2 = 0
  • divisão: 2 / 2 = 1
  • multiplicação: 2 * 4 = 8
  • resto da divisão (ou módulo): 2 % 2 = 0

Onde escrevo meu código?

Se você estiver escrevendo o script dentro do seu arquivo “.html”, seu código deve ser escrito entre as tags <script></script> tanto dentro do <head> quanto no <body>. 

<!DOCTYPE html>
<html>
  <head>
    <!-- Ele pode ficar aqui -->
  </head>
  <body>
    <!-- Ou aqui -->
    <script>
      altert(“Olá!”);
    </script>
  </body>
</html>

Ou ser "importado" de outro arquivo

<!DOCTYPE html>
<html>
  <head>
    <!-- Ele pode ficar aqui -->
  </head>
  <body>
    <!-- Ou aqui -->
    <script src="js/meuScript.js"></script>
  </body>
</html>

Vantagens de um arquivo externo

  • Separar o html e o css do que for código ou regra de negócio
  • O código é mais fácil de ler e manter
  • Pode agilizar o carregamento das suas páginas (performance)

Listas

Existe uma estrutura que nos permite "colecionar coisas".

As listas são estruturas "prontas" que possuem alguns métodos que podemos usar. Os principais são: push, pop, map e reduce. 

let frutinhas = ['banana', 'abacaxi', 'maçã'];

frutinhas.push('pêra');

console.log(frutinhas);
//['banana', 'abacaxi', 'maçã', 'pêra']

Closures

Cuida do "escopo" da variável, evitando que ele seja "perdido".

Em determinadas situações, podemos precisar acessar uma informação mais de uma vez em nosso código. Como as variáveis possuem escopo locais dentro de funções, é possível acessá-las por meio de closures conforme o exemplo:

function multiplicador(a) {
  return function(b) {
    return a * b;
  };
}

const duplica = multiplicador(2);
duplica(3); //6

const triplica = multiplicador(3);
triplica(2); //6
